# Quick Start Guide

## First Time Setup

1. **Open the application**: Double-click `index.html` or open it in your web browser

2. **You're ready!** The system will:
   - Auto-fill today's date
   - Create a blank update for the current week
   - Start tracking immediately

## Your First Week

### Step 1: Fill in the Project Tracker
- Click on any cell in the Description or Details columns to edit
- Select status from the dropdown (Not Started, In Progress, At Risk, Complete)
- Choose due dates using the date picker

### Step 2: Add Tasks to Weekly Checklist
1. Click **"+ Add New Task"**
2. Fill in the task description
3. Select a department/category
4. Set priority (High/Medium/Low)
5. Add optional due date and notes

### Step 3: Add Underwriter Checklist Items
1. Click **"+ Add New Item"** in the Underwriter section
2. Fill in checklist items as needed
3. Check off items when complete

### Step 4: Add Notes
- Use the Additional Notes section for:
  - Meeting summaries
  - Key decisions
  - Banking partner notes
  - Anything else important

### Step 5: Save Your Work
- Click **"Save Update"** button
- Files will download automatically
- Move them to the `archive/` folder

## Starting Your Second Week

1. Click **"New Week"** button
2. Confirm to save current week
3. The system automatically shows last week's checklist in the right panel
4. Click **"Copy Uncompleted Items"** to carry over unfinished tasks
5. Add new tasks for this week
6. Update project tracker
7. Save when done

## Tips

- **Auto-save**: The system saves to browser storage every 2 minutes
- **Filters**: Use department and priority filters to focus on specific tasks
- **Archive**: Visit the Archive page to view all past weeks
- **Search**: Use the search box on the archive page to find specific updates

## Keyboard Shortcuts

- **Tab**: Move to next editable field in table
- **Click to edit**: Any table cell is editable
- **Click away**: Saves automatically

## Need Help?

- Check the main README.md for detailed documentation
- Look at browser console (F12) if something isn't working
- Make sure JavaScript is enabled in your browser
